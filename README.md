# Flutter SDK Example

Flutter 패키지(SDK) 개발 및 사용 예제 프로젝트입니다.

## 프로젝트 구조

```
flutter_sdk_example/
├── calculator_sdk/    # 계산기 SDK 패키지
└── demo/              # SDK를 사용하는 데모 앱
```

## calculator_sdk

간단한 사칙연산 기능을 제공하는 Flutter 패키지입니다.

자세한 내용은 [calculator_sdk/README.md](./calculator_sdk/README.md)를 참조하세요.

## demo

calculator_sdk를 사용하는 예제 앱입니다.

### 실행 방법

```bash
cd demo
flutter pub get
flutter run
```
